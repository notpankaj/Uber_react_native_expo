// ----> create build

1. install eas globally 
   $ npm install -g eas-cli 

2. login to expo ac
   $ eas login

3. build for android
   $ eas build --platform android

4. build for ios
   $ eas build --platform ios
 


 // ----> DEV ENV
 ::::: RUN CODE :::::

 2. npm i  
 3. npm start
